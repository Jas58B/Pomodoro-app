# Pomodoro-app
A customizable Pomodoro timer built with HTML, CSS, and JavaScript. Users can set their own session length, start, pause/resume, and reset the timer. Features a simple and colorful design with coral, pink, and white theme.
## Features

- Start a Pomodoro timer with default 25 minutes.
- Enter a custom session length in minutes.
- Pause and resume the timer at any point.
- Reset the timer to the original or custom session length.
- Clean, beginner-friendly layout inspired by the Codedex tutorial.
- Fully built with HTML, CSS, and JavaScript (no frameworks required).
